#星尘

Stardust  星尘，轻量级服务框架！  
设计理念: 简单，易于维护扩展！  
远程方法调用，服务自动注册和发现，负载均衡，动态伸缩，故障转移，性能监控。  
自认性能还过得去，千万级吞吐率微秒级延迟，因而不支持限流和降级。  
极简设计: 客户端作为组件集成到业务应用，连接后登录，有状态长连接，上报所提供的服务，定时拉取所需要消费的服务列表。调用远程方法时直接从服务列表取得服务提供者地址，并建立长连接。
